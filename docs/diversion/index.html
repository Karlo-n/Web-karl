Diversión API

Descubre nuestra colección de APIs divertidas para añadir entretenimiento y funciones interactivas a tus aplicaciones y servidores de Discord. Desde juegos hasta efectos visuales, estas APIs están diseñadas para crear experiencias memorables.

💋 Kiss

Genera un GIF animado donde un usuario da un beso a otro. Ideal para interacciones sociales divertidas en servidores de Discord o aplicaciones de chat.

Requiere API Key: No | Respuesta: 1-2s
Parámetros: avatar1, avatar2
