📜 Introducción

API Karl ofrece múltiples servicios para bots de Discord, desde herramientas de moderación hasta comandos de entretenimiento.

🔑 Autenticación

Para usar la API, necesitas una API Key. Puedes solicitarla en nuestro panel de control.

🚀 Primeros pasos

Aquí tienes un ejemplo en JavaScript para hacer una petición a la API:

// Ejemplo de petición en JavaScript
fetch('https://apikarl.com/api/ejemplo', {
    headers: { 'Authorization': 'Bearer TU_API_KEY' }
})
.then(res => res.json())
.then(data => console.log(data));

🔹 Endpoints

  • /api/memes – Devuelve imágenes de memes.
  • /api/chistes – Devuelve chistes aleatorios.
  • /api/moderacion – Herramientas de moderación.

❌ Manejo de errores

  • 400 – Solicitud incorrecta.
  • 401 – API Key inválida o faltante.
  • 500 – Error del servidor.
