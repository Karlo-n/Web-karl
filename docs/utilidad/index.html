Utilidades API

Explora nuestra colección de poderosas herramientas de API para mejorar la funcionalidad de tus aplicaciones y servidores de Discord. Estas utilidades están diseñadas para facilitar la integración y ofrecer características avanzadas con solo unas líneas de código.

🚀 Boost Card

Genera una imagen personalizada de un boost para tu servidor con diseño futurista y alta calidad visual. Ideal para destacar a miembros que apoyan tu comunidad.

Requiere API Key: No | Respuesta: 1-2s
Parámetros: avatar, username, background, avatarposicion, usernameposicion
Probar API
🌍 Traductor

Convierte texto entre múltiples idiomas con precisión y rapidez. Elimina barreras lingüísticas en tus aplicaciones y permite comunicación global instantánea.

Requiere API Key: No | Respuesta: 1s
Parámetros: texto, idioma
Probar API
🎉 Bienvenida

Crea impresionantes imágenes de bienvenida personalizadas para nuevos usuarios en tu servidor. Aumenta la retención y crea una primera impresión memorable.

Requiere API Key: Sí | Respuesta: 1-2s
Parámetros: avatar, texto1, texto2, texto3, background (OPCIONAL: color, texto1posicion, texto2posicion, texto3posicion, avatarposicion)
Probar API
📌 Texto a QR

Convierte cualquier texto o URL en un código QR de alta definición para compartir información fácilmente. Ideal para eventos, tarjetas de presentación digitales o enlaces rápidos.

Requiere API Key: No | Respuesta: Instantánea
Parámetros: texto
Probar API
🏆 Carta Ranked

Genera tarjetas de estadísticas de nivel con diseño profesional para sistemas de clasificación. Motiva a los usuarios mostrando su progreso y posición en leaderboards.

Requiere API Key: Sí | Respuesta: 1-3s
Parámetros: avatar, username, xp, maxxp, nivel, background
Probar API
📈 Carta Subida Ranked

Crea imágenes dinámicas que muestran la subida de nivel o rango de un usuario. Celebra hitos y logros con visualizaciones impactantes del progreso.

Requiere API Key: Sí | Respuesta: 1-3s
Parámetros: avatar, username, nivel actual, nivel proximo, background
Probar API
🤖 Roblox Data

Accede a información detallada de usuarios, grupos, inventarios y más elementos de Roblox. Integra datos de Roblox en tus aplicaciones con facilidad y precisión.

Requiere API Key: No | Respuesta: 1-3s
Parámetros: usuario (OPCIONAL: foto=true para capturar pantalla)
Probar API
✅ Verificación Roblox

Sistema completo de verificación para perfiles de Roblox: descripción, inventario, grupos, gamepasses, juegos favoritos y verificación de contacto (email/teléfono) mediante sistema de código seguro.

Requiere API Key: No | Respuesta: 1-2s
Parámetros principales: id (requerido), descripcion, item_id+item_type, gamepass_id, grupo_id, juego_id, juego_favorito_id
Verificación de contacto: contacto+generar_codigo=true o contacto+codigo_verificacion
Probar API
