📌 Documentación API Karl

Aprende a usar la API de Karl para integrar potentes funcionalidades en tu bot.

📜 Introducción

API Karl proporciona múltiples servicios para bots de Discord, permitiendo agregar comandos de diversión, herramientas de moderación y mucho más.

🔑 Autenticación

Para acceder a la API, necesitas una API Key. Solicítala desde el panel de control.

🚀 Primeros pasos

Aquí tienes un ejemplo en JavaScript:

// Ejemplo de petición en JavaScript
fetch('https://apikarl.com/api/ejemplo', {
    headers: { 'Authorization': 'Bearer TU_API_KEY' }
})
.then(res => res.json())
.then(data => console.log(data));

🔹 Endpoints

Lista de endpoints disponibles:

  • /api/memes – Devuelve imágenes de memes.
  • /api/chistes – Devuelve chistes aleatorios.
  • /api/moderacion – Herramientas de moderación para Discord.

❌ Manejo de errores

La API devuelve códigos de error en caso de problemas:

  • 400 – Solicitud incorrecta.
  • 401 – API Key inválida o faltante.
  • 500 – Error del servidor.
